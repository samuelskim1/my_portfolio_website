Projects

Moodify

A Full Stack clone of the popular music streaming service Spotify using React / Redux, Ruby on Rails, JavaScript, AWS, HTML5, and CSS3.

  • Integrated React Player with Redux’s global store and enabled the playbar to detect whether the audio file was part of an album or a playlist. If the current track is part of either, the track would loop properly whenever the end of the album/playlist is met.
  • Implemented efficient/faster postgreSQL queries through properly formatting backend data with Jbuilder which allowed for improved time complexity.
  • Displayed different tracks and albums on my home page on every render utilizing custom Rails backend routes provide a unique user experience every time they visit the home page.
  • Utilized AWS S3 to attach images and audio file urls for songs, tracks, albums, and profiles to my rails backend using ruby’s URI module which allows for scalability of the application and reduced server load.

5tack

A forum built off of the MERN Stack to allow gamers to connect and network using MongoDB, Mongoose, Express, React / Redux, Node, AWS, HTML5, CSS3, and SCSS.

  • Managed a team of 3 software engineers, utilizing efficient git and pull request workflows in order to minimize merge conflicts between the backend and the frontend.
  • Integrated Mongoose’s ability to provide more flexibility with validations and error handling at the application layer as well as efficient drawing data from multiple collections at the same time using Mongoose’s populate method.
  • Utilized SCSS ability of utilizing variables to implement a dynamic navigation bar to display every category and every game associated with each category, allowing for a smooth and interactive user experience.

Spotify Moods

A data visualization of raw audio data that Spotify collects for each of its tracks using JavaScript, D3.js, and Crypto.js.

  • Utilized the D3 library in order to built an app that dynamically renders raw audio data from any Spotify track.
  • Leveraged Crypto.js in order to generate secure tokens to bypass Spotify’s OAuth, enabling me to utilize their API to withdraw raw audio data.
  • Implemented an accordion through DOM manipulation with vanilla javascript and CSS with descriptions of different audio features in order to provide context to users.
